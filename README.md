Windows CF/Ruby Installer Builder
==================================

Requirements
----------------------------------

* Windows 7 or higher
* Ruby 1.9.3
* [Inno Setup](http://www.jrsoftware.org/isinfo.php) 5.5.3 with Inno Setup Preprocessor (ISPP)
* [7-Zip](http://www.7-zip.org/) 9.20


How To Build
------------

```
rake
```
